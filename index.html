Dev Testing Instructions

This page is for internal QA only. Production stores use embed.js, not this file.

  1. Widget: cd apps/widget && npm run dev (port 3002)
  2. SDK: cd asa-sdk-js && npm run build:nudge-dev — not build, build:dev, or build:local
  3. Open this file via Live Server (e.g. :5501) and hard-refresh
  4. Click Nudges Dev Controls → scenario → Apply

Prod-safe: the dev panel only mounts when the SDK bundle is built with npm run build:nudge-dev (NUDGE_DEV=true + 127.0.0.1:3002). Published npm builds point at widget-asa.alphablocks.ai and never activate this UI. The widget also rejects dev commands unless it is served from localhost.
